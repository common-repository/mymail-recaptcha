== Description ==

> This Plugin requires [MyMail Newsletter Plugin for WordPress](https://evp.to/mymail?utm_campaign=wporg&utm_source=MyMail+reCaptcha™+for+Forms)

Adds a reCaptcha™ to your MyMail subscription forms.

You may also try [MyMail Cool Captcha for Forms](https://wordpress.org/plugins/mymail-cool-captcha/)

== Installation ==

1. Upload the entire `mymail-recaptcha` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings => Newsletter => reCaptcha™ and configure the plugin
